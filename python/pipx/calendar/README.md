# Installation

$ ./setup.sh

# Setup vdirsyncer

$ vim ~/.config/vdirsyncer/config

# Run vdirsyncer

$ vdirsyncer discover
$ vdirsyncer sync

# Setup khal

$ khal configure

After finished, khal configuration will be at `~/.config/khal/config`

# Run khal

$ khal calendar

(khal can also be used to create calendar events and sync them to owncloud, add
the commands to do that here)
